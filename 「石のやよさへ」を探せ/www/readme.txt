●RPGツクールMV用のスマホ操作用プラグインです。
デモサイト
http://uchuzine.x0.com/demo/

●ファイル構成
Project　　　　　　　　　
　　├──img　　　　　　　　　　　　
　　│　　└──system　　　　　　　　　　　　
　　│　　　　　├──DirPad.png（方向パッド画像）
　　│　　　　　├──ActionButton.png（アクションボタン画像）
　　│　　　　　└──CancelButton.png（キャンセルボタン画像）
　　└──js　　　　　　　　　　　　
　　　　　└──plugins　　　　　　　　　　　　
　　　　　　　　└──UCHU_MobileOperation.js（メインjs）


●使い方
上記「js」,「img」フォルダの中身をお使いのプロジェクトに格納し
プラグインマネージャーで「UCHU_MobileOperation.js」を
ONにしてください。
※使用するボタン、パッド画像が格納されていないと
Not Foundエラーになります。

各パラメーターの設定方法はプラグインマネージャーより
ヘルプをご確認ください。


●ボタン画像の別素材について
「Button_AnotherMaterial」内にボタン・パッドの
別デザイン素材が用意されています。
こちらも、ご自由にお使いください。

「ActionButton_.png」は
「ActionButton.png」の別案です。
お好きなものをお使いください。
